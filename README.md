# Lead Management System (LMS)

## Project Overview

The Lead Management System (LMS) is a web-based application developed using Django and SQL Server to help organizations efficiently manage leads, products, regions, and related information.

The system provides a centralized platform for storing, updating, searching, and tracking leads generated by a business. It also includes product management, region management, user authentication, and Excel-based bulk product upload functionality.

This project was developed as part of academic learning and practical implementation of Python, Django, Database Management Systems, and Web Development concepts.

# Product Management

The Product Module allows users to manage products available in the system.

### Features

* Add Product
* View Product List
* Edit Product
* Delete Product
* Search Product

### Product Fields

| Field           | Description               |
| --------------- | ------------------------- |
| Product ID      | Unique Product Identifier |
| Product Name    | Name of Product           |
| Category        | Product Category          |
| Is Active       | Active/Inactive Status    |
| Added By        | User who created record   |
| Added Date/Time | Record creation timestamp |

### Validations

* Product Name cannot be empty.
* Product Name must be unique.
* Category must be selected.
* Product ID is auto-generated.

# Product Excel Upload

This functionality allows users to upload multiple products simultaneously using an Excel file.

### Excel Format

| Product Name | Category    | IsActive |
| ------------ | ----------- | -------- |
| Laptop       | Electronics | 1        |
| Mouse        | Electronics | 1        |

### Features

* Bulk Product Upload
* Automatic Product Creation
* Error Reporting
* Duplicate Detection

### Validations

#### Product Name

* Cannot be empty
* Cannot be duplicated

#### Category

* Must exist in Product Category table
* Cannot be empty

#### IsActive

* Defaults to True if empty

### Error Handling

Examples:

* Product already exists
* Category does not exist
* Product name missing

Errors are displayed to the user after upload.

# Installation Steps

### Clone Repository

```bash
git clone https://github.com/siya-nandwani/LMS-Django.git
```

### Navigate to Project

```bash
cd LMS-Django
```

### Create Virtual Environment

```bash
python -m venv myenv
```

### Activate Virtual Environment

```bash
myenv\Scripts\activate
```

### Install Dependencies

```bash
pip install -r requirements.txt
```

### Run Server

```bash
python manage.py runserver
```

### Open Browser

```text
http://127.0.0.1:8000/
```
